HTML에서 a 태그는 "앵커(anchor)" 태그로, 하이퍼링크를 생성하는 데 사용됩니다. 이 태그는 사용자가 클릭할 수 있는 링크를 정의하며, 주로 다른 페이지나 페이지 내의 특정 위치, 또는 이메일 주소 등으로 이동하는 데 사용됩니다.

  1. a 태그 링크 실습

    • 외부 웹사이트로 이동

    • 외부 링크와 내부 링크 섹션으로 이동

    • 주요 속성 섹션으로 이동

    • ktw1398@google.com으로 이메일 보내기

    • 010-2736-2040로 전화 걸기

  2. 외부 링크와 내부 링크

    1. 외부 링크

      • 다른 웹사이트로 이동하는 링크입니다.

      • 현재 웹사이트와는 다른 도메인의 주소를 가지고 있습니다.

      • 전체 URL을 사용하여 지정되며, 다른 웹사이트로 사용자를 이동시킵니다.

      • 예: Visit Example.com

    2. 내부 링크

      • 같은 웹사이트 내의 다른 페이지나 문서를 이동하는 링크입니다.

      • 내부 링크는 동일한 도메인 내의 자원을 가리키며 절대 경로 혹은 상대 경로를 사용하여 지정합니다.

    1. 절대 경로

      • 최상위(ROOT) 디렉터리를 기준으로 경로를 지정합니다.

      • 예: 절대경로로 헤딩 태그 페이지 연결

    2. 상대 경로

      • 현재 페이지를 기준으로 트리 방식으로 경로를 지정합니다.

      • 예: 상대경로"ex01-01-heading.html"로 헤딩 태그 페이지 연결

  3. 주요 속성

    • href : 링크 클릭시 이동할 위치를 지정하는 속성

    • target : 링크가 열릴 창(탭) 이름을 지정. _blank는 새 탭을 지칭

    • mytab에 링크 열기

    • blank에 링크 열기

    • download : 링크에 연결된 자원을 다운로드함.
